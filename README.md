## 🏢 Quiénes somos

**Corporación Maximus 1999, C.A.** es una empresa **distribuidora** de productos al **mayor y al detal**. Nuestro **departamento de tecnología** desarrolla y mantiene las soluciones de software de la empresa — y también presta servicios a **clientes externos**.

## 🚀 Qué hacemos

- **🧩 Implementaciones Odoo** — nuestra especialidad y fuerte principal: configuración, módulos a medida, migraciones y soporte.
- **🌐 Desarrollo web** — sitios y aplicaciones a medida.

## 🛠️ Stack

`Odoo` · `Python` · `PostgreSQL` · `JavaScript (OWL)` · `XML / QWeb` · `HTML / CSS`

## 📐 Cómo trabajamos

Código con **estándares versionados** y **control de calidad automático** (linters + revisión por PR). Nuestro norte es **clean code escalable**: software mantenible, probado y consistente — no atajos.
